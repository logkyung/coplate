# Description
코플레이트는 음식점 리뷰 웹사이트이다. 방문한 음식점에 대해 음식점 이름, 리뷰 제목, 위치(네이버지도 혹은 카카오맵 주소), 별점, 내용을 포함한 게시글을 작성할 수 있다. 

# Environment
- Django
- Python
- HTML
- CSS

# Comment
- 개인 학습으로 강의를 수강하면서 podomarket과 함께 진행했던 프로젝트이다.
- 마찬가지로 Django의 allauth 패키지를 사용해 유저 기능을 구현하는 것에 중점을 두었다.
